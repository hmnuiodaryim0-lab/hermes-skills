# 命理分析实操指南

## 分析前必读

分析前先确认：
1. 求测者**出生年月日时**（精确到分钟）和**出生地**
2. **性别**（男乾女坤，大运方向相反）
3. **求测意图**（综合/事业/感情/健康/单年分析）

---

## 第一问：日主旺衰

**判断方法**：
1. 得令否：出生月份是否为日主当令之月
2. 得地否：地支是否有根（藏干中日主同气）
3. 得助否：天干是否有比劫相助
4. 得势否：全局木火土金水各多少

**旺衰等级**：
- 极旺 → 从旺格
- 中和偏强 → 抑之
- 中和偏弱 → 扶之
- 极弱 → 从格

---

## 第二问：用神

**三步取用**：

**步骤1 — 调候优先**：
冬生（亥子丑月）见火暖局为急用
夏生（巳午未月）见水润局为急用
燥土需水，湿土需火

**步骤2 — 平衡次之**：
日主旺：取官杀克之、或食伤泄之、或财星耗之
日主弱：取印星生之、或比劫助之

**步骤3 — 通关**：
两神相争（如木土交战），用水通关

## 写作原则

1. **不神化**：用现代语言解释，不堆砌玄学术语
2. **有逻辑**：每一步判断都有依据
3. **有出处**：引用典籍原文，注明出处
4. **有边界**：说明命理仅供参考，命运由多因素决定
