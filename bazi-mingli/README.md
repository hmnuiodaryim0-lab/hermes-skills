# 捌字命理研究 · bazi-mingli

基于中国古典典籍的 OpenClaw 八字分析系统。严格遵循「四维交叉验证守则」，以工程化方法论让命理推演透明可控。

---

## 核心特性

- **九本典籍全文**：渊海子平、三命通会、滴天髓、穷通宝鉴、子平真诠、千里命稿、神峰通考、果老星宗、协纪辨方书
- **四维交叉验证**：奠基寻格 → 察气调候 → 寻病抓药 → 跨界落地
- **标准化输出**：七段式报告，含行动建议与风险免责
- **工程化防幻觉**：脚本计算 + 结构化知识库 + Prompt 约束

## 四维交叉验证守则

### 第一维：奠基与寻格（社会属性）
- 《三命通会》查日时组合意象
- 《子平真诠》定格局顺用/逆用

### 第二维：察气与调候（人生品质）
- 《穷通宝鉴》调候寒暖燥湿（优先级高于格局）
- 《滴天髓》验气之流通与阻滞

### 第三维：寻病与抓药（节点定位）
- 《神峰通考》找病药，定大运流年爆发点
- 病药与相神重合处 = 重大事件绝对依据

### 第四维：跨界落地（防伪与现代转化）
- 《果老星宗》星象学交叉验证
- 《协纪辨方书》去神煞伪说
- 《千里命稿》白话翻译落地

## 使用方法

安装后，对 OpenClaw 说：

- "分析一下我的八字"
- "算算命"
- "看看大运走势"
- "流年分析"
- "这八字怎么样"

系统会按四维守则自动调取典籍进行分析。

---

## 标准输出结构

1. **命局定性** — 一句话定义命局核心特征
2. **用神与忌神** — 用神/忌神/调候三栏
3. **命局分析** — 按四维顺序展开，引用典籍原文
4. **流年/大运节点** — 关键年份标出
5. **核心判断** — 直接回答问题，不模糊
6. **行动建议** — 有利方位/月份/当务之急/避免事项
7. **风险提醒与免责声明**

---

## 分析前：两段式访谈

正式分析前需确认：

**第一轮：**
- 想看什么事？（事业/感情/健康/大运/流年）
- 为自己还是为别人？
- 要简报还是详解？

**第二轮：**
- 问流年但没给年份 → 补问具体年份
- 问感情但没说婚姻状态 → 补问已婚/未婚
- 问题泛泛 → "你最想判断哪一个结果？"
